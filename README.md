# Dopamine Detox

Do you remember a single TikTok, Short, or Reel you watched yesterday?

You probably spent 30, maybe 60 minutes scrolling. Maybe more. And you can't recall a single one. That's by design. These feeds aren't built to inform you or entertain you. They're built to keep you swiping. Every clip is disposable. The only thing that matters is that you don't stop.

This extension makes you stop. No timers. No "are you sure?" prompts. No daily limits. It just blocks the feed.

## What It Blocks

- YouTube Shorts
- TikTok
- Instagram Reels

## Install

[Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/dopaminedetox/) 

### From Source

```
git clone https://github.com/aguiarcode/dopamine-detox.git
cd dopamine-detox
npm install
npm run build:firefox    # or: npm run build (Chrome)
```

Load `.output/firefox-mv3/` from `about:debugging#/runtime/this-firefox` or `.output/chrome-mv3/` from `chrome://extensions` in developer mode.
